# reproducible_research_project_g7
**Project title**

Customer Lifetime Value Prediction Using Machine Learning

**Group members**

* Weidong Bai
* Marta Traczyk
* Zuzanna Zofia Herniczek
* Anna Kędzierska-Głodek

**Research question**

How accurately can Customer Lifetime Value (CLV) be predicted using customer payment-related variables, and can alternative modeling approaches improve prediction performance compared to the original study?

**Data source**

The project uses an existing dataset published in the original repository (link to be inserted from the original project source). It contains customer payment-related variables used for CLV prediction. No additional external data collection is planned.

**Planned approach**

We will reproduce the original machine learning pipeline implemented in Python, then rebuild and extend it in R. The workflow will include data preprocessing, feature engineering, and model training using regression-based and machine learning methods. We also plan to compare multiple models and evaluate performance using standard regression metrics, aiming to improve robustness and predictive accuracy over the baseline results.

**Language / tools**

R will be the primary language. We expect to use packages such as tidyverse, caret, randomForest, and possibly xgboost. Python may be referenced only for replication comparison if needed.

**Motivation**

Customer Lifetime Value is a key metric in marketing analytics, and accurate prediction has strong business impact. We are interested in both reproducing an existing machine learning study and improving it using alternative modeling techniques and R-based implementations.
